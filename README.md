# Smart To-Do List (CLI App)

A simple and efficient command-line To-Do List application built using Python.

##  Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using JSON

##  Technologies Used
- Python
- JSON (for data storage)

##  How to Run
1. Make sure Python is installed
2. Run the program:
   python main.py
